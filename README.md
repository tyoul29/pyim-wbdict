# pyim-wbdict
Wubi dicts for pyim
pyim一个替代方案，有以下特点：

1. 提供了`pyim-wbdict-v86`（有词语）和`pyim-wbdict-v86-morphe`（单字纯享）。

2. 由于我是单字用户，崇尚不依赖词库的盲打，所以`pyim-wbdict-v86`版本的顺序是：常用单字、生僻字、词语、符号。

3. 第二候选即是键名，如`s2`为`木`，特殊几个：w八   d石   a戈     x弓。
